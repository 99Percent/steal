metadata

  • typedef
load.metadata

{Object}

 
  • source

An object that is passed between Loader hooks. Any values can be set. These are the ones that steal.js and SystemJS recognize.

Object

Properties

  1. format {String}

    Specifies what type of Syntax is used. This can be specified like:

    "format amd";

  2. deps {Array<moduleName>}

    Dependencies.

  3. exports {String}

    What should be exported.

  4. init {function(Module...)}

    Allows for calling noConflict and for other cleanup. this will be the global object.

    • {Module}

Implemented in SystemJS.
