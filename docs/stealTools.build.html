build

  • function
stealTools.build  
  • source

Build a module and all of its dependencies and optionally other bundles to progressively load.

stealTools.build(config, options)

Parameters

  1. config {Object}

    Specifies configuration values to set on a [System] loader. The main option must be specified. Typically configPath is also specified as that is used to set baseURL. Any System configuration can be specified; however, most other build configuration values are specified in by System.buildConfig in the config file.

    • main {String}

      The module whos dependencies should be built.

    • config {String}

      The path to a configuration file. This will also specify baseURL.

    • baseURL {String}

      If a configuration file is not used, the baseURL value must be set.

    • bundlesPath {String}

      Specifies the path where the production bundles should be placed. Often, this is the same value as System.bundlesPath. By default, the location is "dist/bundles".

      The path can be specified in three ways:

      • Absolute path - bundlesPath starts with / or matches /^\w+:[\/\]/ like: __dirname+"/place" or "c:\my\bundles".
      • Relative to process.cwd() - bundlesPath starts with ./ like "./place".
      • Relative to baseURL - bundlesPath looks like: "packages", "foo/bar".

  2. options {Object}Optional

    Specifies the behavior of the build.

    • minify {Boolean}

      Prevents minification.

    • debug {Boolean}

      Turns on debug messages

    • quiet {Boolean}

      No logging.

    • bundleSteal {Boolean}

      Set to true to include steal in the main bundle.

    • bundle {Array<moduleName>}

      An array of module names that should be progressively loaded.

    • bundleDepth {Number}

      The total number of bundles that need to be loaded for any bundle module.

Returns

{Object}

  • graph {buildGraph}

    A map of moduleNames to node.

  • steal {steal}

    The steal function used to load the main module.

  • loader {Loader}

    The loader used to load the main module.

  • bundles {Array}

    The builds written out.
