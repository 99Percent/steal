System.main

  • property

{String}

 
  • source

String

The main module to load. This is the starting point of the application. In development, the main module is loaded after the @config and @dev modules. In production, only the main module is loaded, but it is configured to load in a bundle.

In production, a System.bundlesPath + System.main bundle is set to include the main module.

Use

Main should be configured by one of the approaches in System.config.
