dest

  • function
 

By default, writes out every module in [baseURL]/dist/global. And adds a ".js" or ".css" to files not ending with ".js" or ".css".

global.dest(loc, path)

By default, writes out every module in [baseURL]/dist/global. And adds a ".js" or ".css" to files not ending with ".js" or ".css".

Parameters

  1. loc {*}

  2. path {String}Optional

    If provided, changes the location where files are written out.
