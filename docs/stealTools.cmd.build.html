stealTools build

  • function
stealTools.cmd.build  
  • source

Builds stealTools from the command line.

stealTools build [--OPTION_NAME OPTION_VALUE]...

Parameters

  1. OPTION_NAME {String}

    Any config or options name in stealTools.build.

  2. OPTION_VALUE {String}

    The value of OPTION_NAME.

Use

StealTools must have been installed into the commandLine like:

npm install steal-tools -g

Then you can run stealTools like:

stealTools build --config app/config.js --main app/app
