BuildOptions

  • typedef
steal-tools.BuildOptions

{Object}

 
  • source

Options used to configure the build process.

Object

Properties

  1. minify=true {Boolean}Optional

    Whether the source code is minified prior to writing.

  2. bundleSteal=false {Boolean}Optional

    Whether StealJS will be included in the built file. Enabling this option will allow you to limit the initial request to just one script.

  3. removeDevelopmentCode=true {Boolean}Optional

    Whether development code (code wrapped with @steal-remove-{start/stop}) will be removed.

  4. ignore {Array<String>}

    Specify modules to not include in the final output. For example if you wanted to load jQuery from a cdn you would provide this option:

    {
  ignore: [ "jquery" ]
}

  5. format {steal-tools.format}

    Specifies a format to transpile the dependency graph to. In the multiBuild all code is automatically transpiled to amd format. In transform the default is to global.
