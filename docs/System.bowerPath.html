System.bowerPath

  • property

{String}

 
  • source

Specifies the path to the folder where Bower dependencies can be found.

String

The folder containing bower dependencies. Defaults to bower_components.

Use

When using the Bower plugin by default it will assume dependencies are located at System.baseURL + /bower_components, which is the default location that Bower installs dependencies. Since this is configurable by Bower itself, bowerPath provices a way to point to the directory where you install Bower deps. Using in the script tag is the best option:

<script src="vendor/steal/steal.js"
  data-bower-path="vendor"
  data-main="main"></script>

Would load the Bower configuration file for, for example, lodash in vendor/lodash/bower.json.

Implementation

Implemented in system-bower.
