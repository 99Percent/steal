pluginify

  • function
stealTools.pluginify  
  • source

A function provided by stealTools.pluginifier that can build out a specific module.

pluginify(moduleName, options)

Parameters

  1. moduleName=config.main {moduleName}Optional

    The module name to build.

  2. options {Object}Optional

    Options that configure how the files are compiled. These options overwrite the pluginifierOptions argument passed to stealTools.pluginifier.

    • ignores {RegExp | Array<RegExp>}

      A JavaScript RegExp or an Array of RegExp's that matches module names that should not be included in the pluginified output.

    • keepDevTags {Boolean}

      By default, removes code in between comments like:

      //!steal-remove-start
REMOVE.THIS;
//!steal-remove-end

      If keepDevTags is true, this code is not removed.

Returns

{String}

The result of moduleName being pluginified.
