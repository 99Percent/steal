stealTools pluginify

  • function
stealTools.cmd.pluginify  
  • source

Pluginifies a module from the command line.

stealTools pluginify [--OPTION_NAME OPTION_VALUE]...

Parameters

  1. OPTION_NAME {String}

    Any config, pluginifierOptions or options name in stealTools.pluginifier or pluginify.

  2. OPTION_VALUE {String}

    The value of OPTION_NAME. The ignores option will be converted to a regular expression.

Returns

{String}

Writes the output to the console.

Use

steal-tools must have been installed into the commandLine like:

npm install steal-tools -g

Then you can run stealTools like:

stealTools pluginify \
           --config app/config.js \
           --main app \
           --ignores foo/.+
           > app.js
